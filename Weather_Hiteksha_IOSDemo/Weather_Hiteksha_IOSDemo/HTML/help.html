Weather App - Help

    • 1. Home screen


          It display list of locations that the user has bookmarked previously.
          You can remove locations from the list.
          Locations can be added by pin on map.
          User can get direction by clicking on pin.
          Display's Known Location at the time of searching, that can be searched.

    • 2. City screen


          User can see forecast with details of : temperature, humidity, rain chances, wind information, Minumum and Maximum temprature, Sunrise and sunset timing and Air Pressure.
          It can be accessed from bookmark list.
          It also shows 5 day's forecast with details of : temperature, humidity, rain chances and wind information.

    • 3. Setting Option


          User can reset bookmark from setting option.
